Enable charger state notification via SMBALERT

  • CommandCode: 14
  • Contains Bit Fields:
    • EN_EQUALIZE_CHARGE_ALERT_BF : Enable alert for lead-acid equalize charge state
    • EN_ABSORB_CHARGE_ALERT_BF : Enable alert for absorb charge state
    • EN_CHARGER_SUSPENDED_ALERT_BF : Enable alert for charger suspended state
    • EN_PRECHARGE_ALERT_BF : Enable alert for precondition charge state
    • EN_CC_CV_CHARGE_ALERT_BF : Enable alert for constant current constant voltage state
    • EN_NTC_PAUSE_ALERT_BF : Enable alert for thermistor pause state
    • EN_TIMER_TERM_ALERT_BF : Enable alert for timer termination state
    • EN_C_OVER_X_TERM_ALERT_BF : Enable alert for C/x termination state
    • EN_MAX_CHARGE_TIME_FAULT_ALERT_BF : Enable max_charge_time_fault alert
    • EN_BAT_MISSING_FAULT_ALERT_BF : Enable alert for missing battery fault state
    • EN_BAT_SHORT_FAULT_ALERT_BF : Enable alert for shorted battery fault state
